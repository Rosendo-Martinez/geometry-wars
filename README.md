# Geometry Wars Game

A PC game based on the 2003 "Geometry Wars" Xbox game.

Tech used:
- C++
- SFML (Graphics Library)
- CMake (Development Tool)
- GCC (Compiler)
- VS Code

## Compiling & Running

Note, these instructions are meant for Ubuntu users.

To remove all binaries:
```
$ make clean
```

To compile program
```
$ make
```
To compile & run program
```
$ make run
```
